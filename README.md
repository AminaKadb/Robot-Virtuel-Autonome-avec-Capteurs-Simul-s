# Robot-Virtuel-Autonome-avec-Capteurs-Simulés
# Description

Ce projet simule un robot autonome se déplaçant dans un environnement virtuel. Le robot détecte des obstacles à l’aide de capteurs simulés et prend des décisions pour avancer, tourner ou éviter les obstacles.

Ce projet met en avant :

- Les compétences en C++ et programmation embarquée.

- La capacité à modéliser un système électronique complet même sans matériel réel.

- La compréhension des algorithmes de contrôle et de décision pour un robot.

# Fonctionnalités

- Déplacement dans une grille 2D (10x10 par défaut).

- Détection d’obstacles via “capteurs virtuels” simulés.

- Algorithme de décision simple : avancer si possible, tourner si un obstacle est détecté.

- Affichage console ASCII du robot et des obstacles.

- Extensible à plusieurs robots ou capteurs supplémentaires.

# Technologies utilisées

- C++ (logiciel embarqué simulé)

- Console ASCII pour visualiser le déplacement
